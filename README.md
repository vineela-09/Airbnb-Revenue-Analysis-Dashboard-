# 🏠 Airbnb Revenue Analysis Dashboard (SQL | Excel | Power BI)

## 📌 Project Overview

Airbnb Revenue Analysis Dashboard is an end-to-end **data analytics project** developed to analyze Airbnb booking data and generate meaningful business insights.

The project focuses on understanding booking trends, revenue performance, occupancy patterns, pricing behavior, and property performance using **SQL, Excel, and Power BI**.

The objective of this project is to transform raw Airbnb booking data into interactive dashboards that help stakeholders optimize pricing strategies, improve occupancy, and maximize revenue.

---

## 🎯 Objectives

- Analyze overall Airbnb revenue performance.
- Understand booking trends and customer behavior.
- Calculate key hospitality performance metrics.
- Identify high-performing properties and locations.
- Analyze occupancy and availability patterns.
- Evaluate pricing strategies using ADR and RevPAR.
- Build an interactive Power BI dashboard for business decision-making.

---
